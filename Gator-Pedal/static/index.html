The Details

What is the Pelican Pedal?

The Pelican Pedal is an all-in-one solution to enhance and augment your current setup. This digital guitar pedal not only provides users with the basic functionality, but it provides them with a community. Users can create profiles and save settings on the pedal for quick access. The Pelican Pedal provides those connected features in order to create a community for users to share their profiles and settings with other players.

The Features

  • Wireless
  • Multiple Mappable Effects
  • Looper
  • Recording, Saving, and Sharing Audio
  • MIDI Support
  • Tilt Effects
  • Sharable User Profiles and Settings
